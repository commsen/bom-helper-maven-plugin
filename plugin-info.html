BOM Helper Maven Plugin

  • | Last Published: 2019-09-27
  • Version: 0.2.0-SNAPSHOT

Plugin Documentation

Goals available for this plugin:

Goal Description
bom-helper:fromJars Creates dependency management entries (BOM) from jar files stored in directory on local file system.
bom-helper:help Display help information on bom-helper-maven-plugin.
Call mvn bom-helper:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
bom-helper:resolve Resolves dependencies declared in dependency management (BOM)
bom-helper:sort Sorts alphabetically the dependencies declared in dependency management (BOM).

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.commsen.maven</groupId>
          <artifactId>bom-helper-maven-plugin</artifactId>
          <version>0.2.0-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.commsen.maven</groupId>
        <artifactId>bom-helper-maven-plugin</artifactId>
        <version>0.2.0-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>
