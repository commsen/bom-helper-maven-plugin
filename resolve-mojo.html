BOM Helper Maven Plugin

  • | Last Published: 2024-02-06

bom-helper:resolve

Full name:

com.commsen.maven:bom-helper-maven-plugin:0.5.0:resolve

Description:

Resolves dependencies declared in dependency management (BOM)

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: validate.

Optional Parameters

Name Type Since Description
typeExtensions Map - Map types to extensions

Parameter Details

typeExtensions:

Map types to extensions
  • Type: java.util.Map
  • Required: No
